Breaks down comments into a number of search engine optimized pages. 


== Description ==

Paginated Comments is a WordPress Plugin ** developed with SEO in mind ** that gives you the ability to break your comments into a number of search engine optimized pages. 

You can create pages based on size of comments (so that no page of comments is greater than a certain file size) or you can create pages based on number of comments per page (100 for example).

Some features include:

* Titles for each page of comments.
* Description and Keywords for each pages of comments.
* Paginate by size of all comments.
* Paginate by a number of comments.
* Personalize your Pretty Permalinks URI's.
* Control page of comments post content.
* Handle this and more features within your WordPress Dashboard with a easily menu.

== Installation ==

1. Unzip the package in an empty directory.
2. Upload the entire directory "paginated-comments" to your Wordpress Plugin Directory, default is: "/wp-content/plugins/".
3. Login to your WordPress Dashboard and go to the plugins section.
4. Activate the Plugin.
